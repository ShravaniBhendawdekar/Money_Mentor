Firo Project Requirements
=========================

Runtime / Environment
- Node.js >= 18
- npm >= 9
- Modern Chromium, Firefox, Safari, or Edge browser

Application Stack
- React 18
- TypeScript 5
- Vite 5

Optional AI Requirement
- Gemini API key for live AI guidance
- Environment variable format:
  VITE_GEMINI_API_KEY=your_gemini_api_key_here
  VITE_GEMINI_MODEL=gemini-2.5-flash

How to install dependencies
- npm install

How to run in development
- npm run dev

How to build for production
- npm run build

How to preview production build
- npm run preview
